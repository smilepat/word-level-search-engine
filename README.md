# word-level-search-engine

영어 단어를 검색하면 **한국 영어 교육과정 기본 어휘(별표3)** 에서 어떤 **급(초등·중등·고등)** 에 해당하는지 찾고,
**굴절어(inflection)만 인정할 때** 와 **파생어(derivation)까지 인정할 때** 의 포함 여부를 각각 판정하는 무빌드 웹 도구입니다.

- **네트워크 0 · 빌드 0 · 설치 0** — `index.html` 을 브라우저로 열면 바로 실행됩니다. (더블클릭 또는 GitHub Pages)
- 데이터: [`Korea-curri-standards-db`](https://github.com/smilepat/Korea-curri-standards-db) 의 교육부 별표3 기본 어휘 (2015 2,977개 · 2022 2,862개).

## 왜 만들었나

교육과정 기본 어휘 목록은 **표제어(headword)** 만 싣고, 그 단어의 변화형은 규정에 따라 처리합니다.

- **굴절어**(복수·3인칭·시제·진행·비교/최상급)는 "같은 단어"로 보아 자동 포함.
- **파생어**(-ly, -ness, -tion, un- 등 접사로 품사·의미가 바뀐 말)는 별도로 판단해야 함.

문제는 원본 DB의 `derivations` 컬럼이 **굴절어와 파생어를 섞어** 담고 있다는 점입니다
(예: `book → books/booked/booking`(굴절) vs `happy → happily/happiness/unhappy`(파생)).
그래서 이 도구는 **형태소 규칙으로 둘을 직접 구분**해 판정합니다.

## 판정 모드

| 모드 | 포함 기준 |
|------|-----------|
| **A · 굴절어만** | 표제어이거나, 표제어의 굴절형이면 포함 |
| **B · 파생어까지** | 위 + 표제어의 파생형도 포함 |

예) `happiness` → 모드 A **미포함**, 모드 B **포함**(happy 경유, 초등).

## 핵심 로직: -er 애매성 해결

`-er`은 **비교급(굴절, tall→taller)** 일 수도, **행위자 명사(파생, teach→teacher)** 일 수도 있습니다.
이 도구는 표제어의 **형제 변이형으로 품사를 추론**해 구분합니다.

- `tall` 은 `tallest` 를 가짐 → 형용사 → `taller` = **굴절어**
- `teach` 는 `teaching` 을 가짐 → 동사 → `teacher` = **파생어**

## 파일 구조

```
index.html        UI (검색창 + 급수/포함 판정표)
engine.js         형태소 판정 엔진 (순수 함수, 무의존)
vocab-data.js     앱용 어휘 데이터 (자동 생성)
vocab-data.json   Node 테스트용 사본 (자동 생성)
engine.test.js    엔진 테스트 (무의존)
prep/
  build-vocab.mjs 원본 CSV → vocab-data.* 생성 스크립트
  source/*.csv    교육과정 어휘 원본 데이터
```

## 사용

```bash
# 1) 실행 — 그냥 index.html 을 브라우저로 열기 (설치 불필요)

# 2) 테스트
node engine.test.js

# 3) 데이터 다시 만들기 (원본 CSV 수정 시)
node prep/build-vocab.mjs
```

## 급 표시 규칙

별표3의 마커를 급으로 변환합니다.

| 마커 | 급 |
|------|-----|
| `*`  | 초등 필수 |
| `**` | 중등 필수 |
| (없음) | 고등 |

## 한계

- 판정은 **철자 기반 형태소 규칙 + 별표3 변이형 목록**에 의존합니다. 품사 정보가 없어 드물게 애매한 경우가 있을 수 있습니다.
- 표제어에 명시된 변이형(97개 표제어)은 목록을 **권위 있는 근거**로 사용하고, 그 외는 규칙으로 일반화합니다.
